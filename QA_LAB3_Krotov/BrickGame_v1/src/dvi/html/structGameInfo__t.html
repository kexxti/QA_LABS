Подробное описание

Основная информация о состоянии игры

Структура передаётся в GUI для отрисовки текущего состояния.

Данные класса

◆ field

int** GameInfo_t::field

Игровое поле HEIGHT×WIDTH (1 — заполнено, 0 — пусто)

◆ high_score

int GameInfo_t::high_score

Рекордный счёт (загружается из файла)

◆ level

int GameInfo_t::level

Текущий уровень (1–10)

◆ next

int** GameInfo_t::next

Следующая фигура: 4 блока + код вращения (5×2)

◆ pause

int GameInfo_t::pause

Состояние паузы: 0 — игра, 1 — пауза, 2 — конец игры, 3 — выход

◆ score

int GameInfo_t::score

Текущий счёт

◆ speed

int GameInfo_t::speed

Скорость (не используется напрямую, вычисляется по уровню)
